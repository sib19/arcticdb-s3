  METHOD 1: HTTPS (Easiest)
  ─────────────────────────

  git branch -M main
  git push -u origin main

  Best for: Quick setup, first-timers
  Where: All guides cover this


  METHOD 2: SSH (Most Secure)
  ──────────────────────────
  git remote add origin git@github.com:YOUR_USERNAME/arcticdb-project.git
  git branch -M main
  git push -u origin main

  Best for: Frequent pushes, security
  Where: GITHUB_UPLOAD_GUIDE.md (Method 2)


  METHOD 3: GitHub CLI (Fastest)
  ──────────────────────────────
  cd arcticdb-project

  Best for: One-command setup
  Where: GITHUB_UPLOAD_GUIDE.md (Method 3)
